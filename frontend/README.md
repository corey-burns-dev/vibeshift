# Frontend (Vite + React + TanStack Query)

Frontend application for Sanctum, built with React + TypeScript + Vite.

## Quick Start

From repo root:

```bash
cd frontend
bun install
bun run dev
```

Dev server default: `http://localhost:5173`

## Useful Commands

```bash
bun run build
bun run test
bun run lint
bun run format
```

For repo-wide rules and constraints, see `/AI.md`.
